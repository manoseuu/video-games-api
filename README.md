# Video Games API

This project is a RESTful API developed using Node.js, Express, and MongoDB.
It was created as part of an academic assignment.

## Features
- User registration and login
- Video games CRUD operations
- MongoDB Atlas integration
- Password hashing with bcrypt
- JSON-based API responses

## Technologies
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- bcrypt
- Postman (for testing)
